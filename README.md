## snowflake-java-client

maven依赖:

```xml
<dependency>
    <groupId>com.github.yingzhuo</groupId>
    <artifactId>snowflake-java-client</artifactId>
    <version>1.0.2</version>
</dependency>
```

`spring-boot`配置

```yaml
snowflake:
  response-type: json
  hostname: "localhost"
  port: 18080
```

直接使用`com.github.yingzhuo.snowflake.SnowflakeUtils`即可。
